```sql
CREATE TABLE public.collections (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  user_id UUID NOT NULL REFERENCES public.profiles(id) ON DELETE CASCADE,
  name TEXT NOT NULL,
  description TEXT,
  created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
  updated_at TIMESTAMPTZ NOT NULL DEFAULT now()
);

CREATE TABLE public.collection_items (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  collection_id UUID NOT NULL REFERENCES public.collections(id) ON DELETE CASCADE,
  generation_id UUID NOT NULL REFERENCES public.generations(id) ON DELETE CASCADE,
  added_at TIMESTAMPTZ NOT NULL DEFAULT now(),
  UNIQUE(collection_id, generation_id)
);
```

2. Create `src/lib/collections.ts` with these exported functions:

   - `fetchCollections(userId: string): Promise<Collection[]>` — Fetches all collections for user from `collections` table, ordered by `created_at DESC`. For each collection, also fetches item count via a separate query: `supabase.from('collection_items').select('id', { count: 'exact', head: true }).eq('collection_id', col.id)`. Maps to frontend type.

   - `createCollection(userId: string, name: string, description?: string): Promise<Collection>` — Inserts into `collections` table with `user_id`, `name`, `description` (null if not provided). Returns created collection with itemCount: 0.

   - `deleteCollection(collectionId: string): Promise<void>` — Deletes the collection row. CASCADE will remove collection_items too.

   - `addToCollection(collectionId: string, generationId: string): Promise<void>` — Inserts into `collection_items` with `collection_id` and `generation_id`. Use `.upsert()` to handle duplicate gracefully (the UNIQUE constraint).

   - `removeFromCollection(collectionId: string, generationId: string): Promise<void>` — Deletes from `collection_items` where `collection_id` and `generation_id` match.

   - `fetchCollectionItems(collectionId: string): Promise<string[]>` — Returns array of `generation_id` strings for a given collection. Query: `supabase.from('collection_items').select('generation_id').eq('collection_id', collectionId)`.

3. Create `src/hooks/useCollections.ts`:

   - Export `useCollections()` hook with React state:
     - `collections: Collection[]` — list of user's collections
     - `loading: boolean` — true during initial fetch
     - `activeCollectionId: string | null` — which collection is being filtered (null = show all)
     - `activeCollectionItemIds: string[]` — generation IDs in the active collection

   - On mount (when user exists from `useAuth()`), calls `fetchCollections(user.id)` and sets state.

   - Exposes methods:
     - `create(name: string, description?: string)` — calls `createCollection`, adds to state
     - `remove(collectionId: string)` — calls `deleteCollection`, removes from state, resets activeCollectionId if it was the deleted one
     - `addItem(collectionId: string, generationId: string)` — calls `addToCollection`, increments itemCount in local state, adds to activeCollectionItemIds if same collection
     - `removeItem(collectionId: string, generationId: string)` — calls `removeFromCollection`, decrements itemCount, removes from activeCollectionItemIds
     - `setActiveCollection(collectionId: string | null)` — sets activeCollectionId. If not null, calls `fetchCollectionItems(collectionId)` and sets `activeCollectionItemIds`.

   - Returns `{ collections, loading, activeCollectionId, activeCollectionItemIds, create, remove, addItem, removeItem, setActiveCollection }`

   - Add `filterCollectionId: string | null` to interface and initial state (null)
   - Add `setFilterCollectionId: (id: string | null) => void` action
   - This allows Canvas to filter displayed generations by collection
