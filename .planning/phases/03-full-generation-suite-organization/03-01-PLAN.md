1. Create `src/lib/generations.ts` with three exported functions:

   - `saveGeneration(userId: string, gen: Generation): Promise<void>` — Upserts a generation row to Supabase `generations` table. Maps from the frontend `Generation` type to DB columns: `id` (use gen.id), `user_id` (userId), `mode` (gen.mode), `prompt` (gen.prompt), `parameters` (JSON of gen.params), `image_url` (gen.imageUrl), `thumbnail_url` (gen.thumbnailUrl), `status` (gen.status), `error_message` (gen.errorMessage), `created_at` (gen.createdAt). Use `supabase.from('generations').upsert(...)` so it works for both insert and update.

   - `fetchGenerations(userId: string): Promise<Generation[]>` — Fetches all generations for a user from Supabase, ordered by `created_at DESC`, limit 200. Maps DB rows back to the frontend `Generation` type: `id`, `mode` (cast to GenerationMode), `prompt`, `imageUrl` (from image_url), `thumbnailUrl` (from thumbnail_url), `status` (cast to GenerationStatus), `errorMessage` (from error_message), `params` (from parameters JSONB, with fallback defaults for missing fields), `createdAt` (from created_at ISO string).

   - `fetchGenerationById(genId: string): Promise<Generation | null>` — Fetches a single generation by ID. Same mapping as above. Returns null if not found.

2. Create `src/hooks/useGenerations.ts`:

   - Export `useGenerations()` hook that:
     - Gets `user` from `useAuth()` hook
     - Gets `generations`, `addGeneration`, `updateGeneration` from workspace store
     - On mount (when user exists), calls `fetchGenerations(user.id)` and populates the store with `set({ generations: result })` via a new `setGenerations` action
     - Exposes a `persistGeneration(gen: Generation)` function that calls `saveGeneration(user.id, gen)` (fire-and-forget, log errors to console)
     - Returns `{ generations, persistGeneration, loading }` where loading is true during initial fetch

3. Add `setGenerations: (gens: Generation[]) => void` to workspace store:
   - In `src/stores/workspaceStore.ts`, add to the interface: `setGenerations: (gens: Generation[]) => void`
   - Implement as: `setGenerations: (generations) => set({ generations })`
